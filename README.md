# YT_downloder
This is e extension that you have to add in your Chrome/Brave browser to download YouTube video

How to use 
1) Download the "tubly_downloader_v1.5.3.zip" file
2) Extract it (any whare you want)
3) in your browser's url section search "chrome://extensions/"
4) Open Developer mode you can find it on Top right corner (most probably).
5) Now drag and drop the Extracted file in this tab
6) Pin the Extension
7) Have fun <3
